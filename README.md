# p0-w3-d3-t3 · Дерево комментариев

> Тип: рекурсия · Репозиторий: один отдельный

> Подключить файл, отсмотреть в браузере.

## Задача

Комментарии под постом — это дерево: у комментария есть ответы, у ответов свои ответы, и так на любую глубину. Обойти такую структуру циклом нельзя (глубина заранее неизвестна) — нужна рекурсия, которая спускается по вложенности.

```
дерево комментариев (ответы вложены в ответы)
  коммент
   ├─ ответ
   │   └─ ответ на ответ
   └─ ответ
рекурсия: посчитать все комменты и максимальную глубину
```

## Что нужно сделать

В `comments.js` взять готовое дерево комментариев (ниже). Написать рекурсивную функцию, которая считает общее число комментариев (вместе со всеми вложенными), и вторую (или ту же) — которая находит максимальную глубину вложенности.

Входные данные:

```js
const thread = {
  text: "Отличная статья",
  replies: [
    {
      text: "Согласен",
      replies: [
        { text: "И я", replies: [] },
        { text: "А мне не зашло", replies: [
          { text: "Почему?", replies: [] },
        ]},
      ],
    },
    { text: "Спасибо автору", replies: [] },
  ],
};
```

Ожидаемый результат:

```
Всего комментариев: 6
Максимальная глубина: 4
```

(глубина считается по уровням: корневой комментарий — уровень 1)

## Критерии приёмки

* Рекурсивная функция корректно считает все комментарии на любой глубине; есть базовый случай (пустые `replies`).
* Определяется максимальная глубина дерева.
* Ты можешь объяснить, что здесь базовый случай, что рекурсивный, и почему цикл тут не подошёл бы.
